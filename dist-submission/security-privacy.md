# Security & Privacy

## Data collected
- None by default (analytics events are local-only placeholders and disabled)

## Data storage location
- Local vault/plugin data only (no remote service by default)

## Deletion and uninstall behavior
- Uninstall/removal removes generated package files from your environment
- Removing the plugin disables features; local plugin data remains in your vault until manually deleted

## Security controls
- Use least-privilege scopes and rotate credentials in production.
- Run dependency and vulnerability checks before each release.
- Keep support and security contact channels current.
